# 💻 Machine Learning Portfolio

Welcome! This repository showcases my Machine Learning projects. Each project covers a different problem type and progressively builds on more advanced techniques.

---

## 🧪 Projects

### 1️⃣ [House Price Prediction (Regression)](https://github.com/Inam54/House-Price-Prediction)
Predict house prices using **Linear Regression** and **Decision Tree Regressor**.
Includes data preprocessing, feature scaling, and encoding.

### 2️⃣ [Student Performance Prediction (Regression)](https://github.com/Inam54/Student-Performance-Prediction)
Predict students' average scores using **Decision Tree** and **Random Forest Regressors**.
Includes one-hot encoding, scaling, and feature importance analysis.

### 3️⃣ [Customer Churn Prediction (Classification)](https://github.com/Inam54/House-Price-Prediction)
Predict whether a customer will churn using **Decision Tree** and **Random Forest Classifiers**.
Includes handling categorical data, scaling, and imbalanced datasets.

### 4️⃣ [Income Prediction System (Classification)](https://github.com/Inam54/Adult-Income-Prediction)
Predict whether an individual's income exceeds $50K per year using **Logistic Regression**.
Includes OOP pipeline, encoding, and scaling.

### 5️⃣ [BMW Sales Classification (Classification)](https://github.com/Inam54/BMW-Sales-Prediction)
Predict whether BMW sales are **high or low** using **Logistic Regression**, **Decision Tree**, and **Random Forest Classifiers**.
Includes OOP pipeline, one-hot encoding, scaling, and handling imbalanced datasets.

### 6️⃣ [AI Salary Prediction System — Capstone Project 1 (Regression)](https://github.com/Inam54/Capstone-Project-1-AI-Salary-Predictor)
A full **end-to-end ML application** that predicts AI job salaries based on 23 features including job role, experience level, country, industry, and economic indicators.
Includes:
- Complete **ML pipeline** with preprocessing, training, tuning, and automatic best model selection
- **Random Forest (Tuned)** selected as best model based on R² Score
- **FastAPI** backend with `/predict` and `/health` endpoints
- **Pydantic v2** custom field validators and cross-field validation
- **Streamlit** frontend UI for real-time predictions
- Clean **JSON responses** for both predictions and errors
